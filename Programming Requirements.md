# Programming Requirements Document (PRD)

## Purpose

EcoDispose allows safe and legal disposal of electronic devices.

## Users

- IT technicians
- Recycling staff
- General public

## Requirements

- Simple menu system
- Device tracking
- Waste classification
- Secure wipe recording

## Platform

- Python 3
- Command-line interface

## Security

- Data wipe confirmation
- No personal data stored

## Problem

Many organisations fail to track e-waste properly, leading to environmental harm and data risks.

## Solution

EcoDispose provides a structured system for managing disposal safely and efficiently.
