# EcoDispose ♻️

EcoDispose is a Python-based system designed to support safe and WEEE-compliant disposal of electronic devices.

## 📌 Features

- Device registration system
- Waste classification (Reuse / Recycle / Dispose)
- Secure data wipe confirmation
- Privacy policy display
- Multi-device handling

## 🧠 Purpose

The system helps organisations and individuals manage electronic waste responsibly while ensuring data security and compliance with UK regulations.

python main.py
